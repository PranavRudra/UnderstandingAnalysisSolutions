## (c)

False. Let

```math
\begin{align*}
&A &&= \{ 1 \} \\
&B &&= \{ 1 \} \\
&C &&= \{ 1, 2 \}
\end{align*}
```

Then,

```math
A \cap (B \cup C) = \{ 1 \} \cap (\{ 1 \} \cup \{ 1, 2 \}) = \{ 1 \} \cap \{ 1, 2 \} = \{ 1 \}
```

However,

```math
(A \cap B) \cup C = (\{ 1 \} \cap (\{ 1 \}) \cup \{ 1, 2 \}) = \{ 1 \} \cup \{ 1, 2 \} = \{ 1, 2 \}
```

So the identity doesn't hold.
